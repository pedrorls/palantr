Palantr API Reference

Base URL `palantr.herokuapp.com/api/v1/`

Endpoint Method Example Sample Response Description
/topics/ GET http://palantr.herokuapp.com/api/v1/topics/ 
            [
              { id: 1,
                title: 'Hi!',
                categories: 'Computer, Friends',
                content: 'Post about Friends'
              },
              {
                id: 2,
                title: 'New Post',
                categories: 'Candy',
                content: 'Post about Candy'
              }
            ]
Fetches all the topics registered.
/topics/:topic_name GET http://reduxblog.herokuapp.com/api/v1/topics/:topic_name 
            {
              id: 1,
              title: 'Hi!',
              categories: 'Computer, Friends',
              content: 'Blog post content'
            }
Returns a single topic if it exists.
/topics/:topic_name/posts GET http://reduxblog.herokuapp.com/api/v1/topics/:topic_name/posts 
            {
              id: 1,
              title: 'Hi!',
              categories: 'Computer, Friends',
              content: 'Blog post content'
            }
Fetches all the posts related with the given topic.
